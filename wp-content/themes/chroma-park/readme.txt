A retro responsive design with a header menu, two sidebars, and color customization.

== Description ==

Chroma Park is a retro responsive design that gives your blog a late 2000s look.
Built on Green Park, a theme from the late 2000s, Chroma Park has added features not present in Green Park.
Chroma Park features two optional sidebars and an optional header menu. The header, footer, content, and sidebars can have their colors customized individually.
